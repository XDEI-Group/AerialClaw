# change_altitude — 相对高度调整

## 使用时机
需要在当前位置升高或降低时使用。

## 参数
- delta: float，相对高度变化（米）
  - delta=20 → 从当前位置升高20米
  - delta=-10 → 从当前位置下降10米
  - delta=50 → 从当前位置升高50米

## 前提条件
- 无人机在空中

## 执行流程
1. 获取当前高度，必须先执行get_position技能，否则调整高度
2. 计算目标高度 = 当前高度 + delta
3. 纯垂直运动到目标高度，水平位置不变
4. 到达后悬停

## 注意事项
- 正数升高，负数下降
- 最低安全高度 2m，不会降到 2m 以下
- 这是相对变化，不是绝对高度
- 水平位置完全不变

## 输出
- previous_altitude: 变化前的高度（米）
- current_altitude: 变化后的高度（米）
- change: 变化量描述
