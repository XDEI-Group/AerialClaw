# ask_user -- 主动提问

## 使用时机
遇到不确定的情况，需要人类判断时使用。
例如：发现不确定的目标、不知道是否该继续前进、需要确认任务范围。

## 参数
- question: str, 要问操作员的问题（清晰简洁）

## 执行流程
1. 将问题推送给操作员
2. 等待操作员回答（最多60秒）
3. 返回操作员的回答
4. 如果超时，返回"(操作员未回答，自行判断)"

## 注意事项
- 问题要具体、简洁，方便操作员快速回答
- 不要问太多问题，尽量自主判断
- 如果超时未回答，根据最安全的选项自行决定
- 适合问是否/选择类问题，不适合问开放性问题

## 输出
- answer: 操作员的回答
