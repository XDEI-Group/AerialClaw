# update_map -- 更新场景地图

## 使用时机
探索新区域、发现新地标时使用。
将发现的建筑/设施/地形信息追加到 WORLD_MAP.md。

## 参数
- landmark_name: str, 地标名称（简短有辨识度）
- description: str, 外观描述（颜色、形状、大小等特征）

## 执行流程
1. 记录当前无人机位置作为地标坐标
2. 追加到 WORLD_MAP.md 的"探索发现"段落
3. 下次任务时 LLM 可以参考这些信息做更好的规划

## 最佳实践
- 只记录有辨识度的地标，不要记录每一个小物体
- 描述要包含视觉特征（颜色、形状、大概高度）
- 命名要有意义（如"蓝色玻璃高楼"而不是"建筑1"）

## 输出
- updated: 是否更新成功
- total_landmarks: 地图中总地标数
