# 搜救策略

## 概述
在灾区搜索受困者, 标记位置, 汇报给操作员。

## 推荐流程
1. 起飞到安全高度(10-12m), 高于废墟但能看清地面
2. 飞到灾区上空(右后方约50m)
3. 原地旋转一圈(look_around), 先大致了解环境
4. 用fly_relative沿着灾区做"割草机"式搜索:
   - 往前飞20m → 原地观察 → 往右平移15m → 往回飞20m → 再往右平移15m...
   - 每次悬停观察时, 注意LiDAR有没有异常近的目标
5. 发现疑似目标时:
   - hover悬停
   - observe(focus="寻找地面上的受困者") 仔细看!
   - 如果确认有人, mark_location标记
6. 搜索完成后:
   - 用get_marks列出所有发现
   - 返回起飞点(return_to_launch)
   - 向操作员汇报发现了几个目标、分别在哪

## 注意事项
- 电量低于40%时立刻返航, 不要冒险
- 灾区有高5-8m的废墟, 别飞太低
- 连续搜索中如果找不到目标, 换个高度或角度再试
- 每发现一个目标就标记, 别等到最后再标记

## 从经验中学习
- 如果之前搜索过这个区域, 查看MEMORY里的记录
- 用上次的经验改进路线: 跳过已确认空旷的区域, 重点搜索上次漏掉的地方
- 如果某个搜索模式效果好(比如螺旋搜索), 下次优先用它
