# 目标搜索策略

## 概述
在指定区域搜索特定目标(人员/物体), 标记发现位置。

## 搜索模式
### 割草机式(推荐)
从区域一端开始, 左右往返推进:
```
→→→→→→→→→
          ↓
←←←←←←←←←
↓
→→→→→→→→→
```
每排间距15-20m, 确保不遗漏。

### 螺旋式
从外围向中心收缩:
适合知道目标大致在某个范围但不确定精确位置。

### 定点观察
飞到某个位置上空, hover + look_around。
适合已经知道大致方位的情况。

## 推荐流程
1. 飞到搜索区域边缘上空
2. 选择搜索模式(割草机最稳妥)
3. 每飞一段就hover观察
4. 发现目标立刻mark_location
5. 搜索完毕后汇报: 找到几个, 分别在哪

## 注意事项
- 搜索高度别太高, 10-12m看得最清
- 每次hover时多等几秒, 让传感器充分采集
- 如果LiDAR检测到异常近的物体, 可能就是目标
