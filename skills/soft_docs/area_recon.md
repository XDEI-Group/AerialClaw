# 环境侦察 (Area Recon)

## 触发条件
当你需要对一片区域进行快速了解，掌握周围环境全貌时使用。

## 策略流程

1. **确认位置和高度**: `get_position` 获取当前坐标和高度
2. **环视四周**: `observe(direction="around")` 拍摄所有方向
3. **report 汇报**: 把观察结果汇报给操作员
4. **update_map 更新地图**: 发现新地标就记录
5. **如果信息不足**: 用 `fly_relative` 换位置/高度，再观察
6. **主动 perceive**: 对感兴趣的方向用自定义问题深入分析

## 决策树
```
当前高度 < 50m?
  → change_altitude(60) 先升高获得全局视野
当前高度 >= 50m?
  → observe(around) 环视
  → 发现目标? → fly_relative 接近 → 降低高度 → 详细观察
  → 没发现? → fly_relative 移动 100m → 再次环视
```

## 关键原则
- **先高后低**: 高空看全局，低空看细节
- **先远后近**: 先了解大范围，再聚焦小区域
- **持续记录**: 每次观察都 report + update_map
- **灵活调整**: 根据看到的内容决定下一步，不要死板
