# WORLD_MAP.md — 场景物体清单 (颜色+形状+坐标)

## 坐标系
NED (North-East-Down) 本地坐标, 原点=起飞点(0,0,0)
- North(北): NED X 正方向
- East(东): NED Y 正方向
- Down(下): NED Z 正方向 (高度用负值, z=-15 = 15m高)

fly_to 参数: [North, East, Down], 例如 [-30, 40, -12] = 南30m 东40m 高12m
