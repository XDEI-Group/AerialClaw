# WORLD_MAP.md — 上海城市场景

## 坐标系
NED (North-East-Down) 本地坐标, 原点=起飞点(0,0,0)
- North(北): NED X 正方向
- East(东): NED Y 正方向  
- Down(下): NED Z 正方向 (高度用负值, z=-15 = 15m高)

fly_to 参数: [North, East, Down], 例如 [-30, 40, -12] = 南30m 东40m 高12m

## 起飞点
位于上海陆家嘴金融区东侧的空旷地面, 面朝城市天际线和水面。

## 周围环境概览

### 近处 (100m 内)
- 南侧约 40m: 低层建筑群 (约14m高)
- 正前方(北偏西): 几栋中层建筑 (10-20m)

### 中距离 (100-200m)
- 东北方向 (~110m): 高层建筑群, 最高约150m (可能是商业区)
- 西南方向 (~100m): 中层建筑群 (16-24m)

### 远处 (200-400m)
- 西方偏南 (~250m): 超高层摩天楼群, 最高约477m (上海金融区三件套方向)
  - 位于约 NED (-60, -140) 方向
- 西北方向 (~250m): 约328m 高的塔形建筑群

### 水面
- 东侧方向有水面 (可能是黄浦江)

## 飞行安全提示
- 起飞点周围 40m 内有低层建筑, 起飞后立刻升高到 20m 以上
- 向西飞行时注意超高层建筑 (最高477m), 保持安全距离或足够高度
- 建议巡航高度: 30-50m (能看清地面细节, 又避开大部分建筑)
- 靠近摩天楼群时升高到 100m 以上
- 东侧水面上空无障碍, 适合安全飞行

## 建议巡检路线
从起飞点出发, 顺时针绕一圈:
1. 起飞到 30m → 向东飞到水边 (E+100m)
2. 沿水面向北飞 (N+150m) 
3. 向西飞向摩天楼群方向 (W-200m), 升高到 80m
4. 沿摩天楼群外围向南飞
5. 返回起飞点降落
