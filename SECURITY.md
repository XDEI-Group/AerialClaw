# Security Policy

AerialClaw controls simulated and potentially physical aerial systems. Please treat safety issues as security issues.

## Reporting

Open a private security advisory on GitHub or contact the maintainers through the XDEI Group project page.

## Safety expectations

- Do not bypass flight safety envelopes.
- Do not hard-code API keys or credentials.
- Do not enable arbitrary file/network access without sandboxing and audit logs.
- Treat real-drone operation as experimental until separately validated.
